## 👨‍💻 About Me

I'm Bilal Azam, an Offensive Security Researcher and Bug Bounty Hunter focused on finding real-world vulnerabilities in web applications and APIs.

I don't just run tools — I understand how applications work, where they break, and how to exploit those breaks before someone else does.

- 🎯 **Goal:** Become a world-class Web Security Researcher
- 🌐 **Education:** BS Cybersecurity
- ⚡ **OS:** Kali Linux / Parrot OS / Windows
- 🐧 **Linux Power User** — terminal is my home
- 💼 Open to security collaborations, bug bounty programs, and pentest roles
